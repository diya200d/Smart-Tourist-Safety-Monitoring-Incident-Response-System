# Smart Tourist Safety Monitoring & Incident Response System

**Overview**
- Full‑stack MERN + Electron application
- Helps tourists stay safe by reporting incidents and receiving alerts
- Admins and safety officers can monitor incidents and manage responses

**Key Features**
- **Authentication**: Admin,Tourist roles
- **Dashboard**: Real‑time safety statistics and activity monitoring
- **Incident Management**: Report, track, and resolve incidents with severity levels
- **Location Management**: Add and manage tourist locations with GPS support
- **Safety Alerts**: Create and manage alerts for weather, traffic, medical, and security
- **User Management**: Admins can manage accounts, roles, and activity
- **Desktop Application**: Cross‑platform app built with Electron

**Technology Stack**
- **Frontend**: React 18, React Router, Styled Components, Axios
- **Backend**: Node.js, Express.js, MySQL, JWT, Bcrypt
- **Desktop**: Electron, Electron Builder

**Setup Instructions**
- Clone repository: `git clone https://github.com/diya200d/Smart-Tourist-Safety-Monitoring-Incident-Response-System.git`
- Install dependencies:
  - Root: `npm install`
  - Backend: `cd backend && npm install`
  - Frontend: `cd ../frontend && npm install`
- Configure `.env` in backend with database and JWT settings
- Start servers: `npm run dev`

**Demo Flow**
- **Tourist**: Register/login, report incidents, receive alerts
- **Admin**: Manage users, monitor dashboard, create alerts, track incidents

**Future Enhancements**
- Real‑time notifications with WebSocket
- Mobile companion app
- AI‑powered risk assessment
